# AAMN Assets

This directory contains visual assets for the AAMN project.

## Files

| File | Description | Usage |
|------|-------------|-------|
| `logo.svg` | Main logo (512x512) | Website, presentations |
| `icon.svg` | Small icon (256x256) | App icon, favicon |

## Colors

| Color | Hex | Usage |
|-------|-----|-------|
| Primary | `#e94560` | Accent, highlights |
| Dark Blue | `#1a1a2e` | Background |
| Navy | `#0f3460` | Secondary elements |
| Light Navy | `#16213e` | Gradients |
